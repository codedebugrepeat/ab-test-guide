# Heading style

All headings (h1–h6, page titles in `chapters.ts`, section titles in nav) use **American English sentence case**: capitalize only the first word and proper nouns. Never use title case.

- Correct: `Why small samples lie`, `Case study`, `The signal-to-noise problem`
- Wrong: `Why Small Samples Lie`, `Case Study`, `The Signal-To-Noise Problem`

Apply this rule when writing or reviewing any new page, section, or navigation label.
